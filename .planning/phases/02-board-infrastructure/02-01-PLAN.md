2. Create `components/chess/useChessGame.ts` — a custom hook encapsulating chess.js game state:

```typescript
'use client';

import { useState, useCallback, useMemo } from 'react';
import { Chess, Square } from 'chess.js';

export interface ChessGameState {
  fen: string;
  selectedSquare: Square | null;
  legalMoves: Square[];
  selectSquare: (square: Square) => void;
  makeMove: (from: Square, to: Square) => boolean;
  reset: (fen?: string) => void;
  game: Chess;
}
```

The hook should:
- Accept an optional initial FEN string parameter (default: starting position)
- Store a `Chess` instance in state (created with `new Chess(fen)`)
- Track `selectedSquare: Square | null` in state
- Compute `legalMoves: Square[]` — when a square with a piece of the current turn's color is selected, call `game.moves({ square, verbose: true })` and extract the `.to` fields
- `selectSquare(square)`:
  - If square has a piece of the side to move → set as selectedSquare, compute legalMoves
  - If square is in legalMoves (a valid destination) → call makeMove(selectedSquare, square)
  - If square is empty and nothing selected → do nothing
  - If tapping the already-selected square → deselect (clear selectedSquare and legalMoves)
- `makeMove(from, to)`: call `game.move({ from, to })`, update FEN state, clear selection, return true/false
- `reset(fen?)`: create new Chess instance with given FEN or default, clear selection
- Return `{ fen, selectedSquare, legalMoves, selectSquare, makeMove, reset, game }`

Note: The hook should use `useState` for `game` instance so React tracks it. When game state changes after a move, create a new Chess instance from the resulting FEN to ensure immutability, OR use a ref for the Chess instance and a separate FEN state for triggering re-renders. Prefer the ref approach:
- `const gameRef = useRef(new Chess(initialFen))`
- `const [fen, setFen] = useState(initialFen)`
- After moves, call `setFen(gameRef.current.fen())` to trigger re-render

Export the hook as a named export: `export function useChessGame(initialFen?: string): ChessGameState`

```typescript
interface ChessBoardProps {
  initialFen?: string;
  boardWidth?: number;          // override width, otherwise responsive
  onMove?: (from: string, to: string) => void;  // callback after successful move
  gameRef?: React.MutableRefObject<ChessGameState | null>;  // expose game state to parent
}
```

Implementation details:

a. **Responsive sizing (D-01, D-02, D-03, D-04, BOARD-02):**
   - Use a state variable `boardWidth` computed from container width
   - Use a `useRef` + `ResizeObserver` on the outer Box to measure available width
   - Clamp: `Math.min(containerWidth, 480)` for max, minimum is `56 * 8 = 448px`
   - If prop `boardWidth` is provided, use it directly (skip responsive)
   - Pass the computed width to `<Chessboard boardWidth={computedWidth} />`

b. **RTL isolation (D-12, D-13, D-14, BOARD-05):**
   - Outer wrapper Box gets: `sx={{ direction: 'ltr', width: 'fit-content', margin: '0 auto' }}`
   - This prevents the RTL locale from flipping the board grid
   - Board always shows white at bottom, a-file on left (standard orientation)

c. **Tap interaction (D-15, D-16, D-17, D-18, BOARD-03, BOARD-04):**
   - Use the `useChessGame` hook for state
   - Pass `arePiecesDraggable={false}` to Chessboard (per D-18)
   - Pass `onSquareClick={(square: Square) => gameState.selectSquare(square)}` to handle taps
   - If `gameRef` prop provided, assign `gameRef.current = gameState` in a useEffect so parent can access

d. **Move highlight styles (D-08, D-09, D-10, D-11):**
   - Compute `customSquareStyles` object from `gameState.selectedSquare` and `gameState.legalMoves`:
     - Selected square: `{ [selectedSquare]: { backgroundColor: 'rgba(255, 255, 0, 0.4)' } }` (yellow tint)
     - Legal move squares (non-capture): `{ [sq]: { background: 'radial-gradient(circle, rgba(0,128,0,0.4) 25%, transparent 25%)', borderRadius: '50%' } }`
     - Legal capture squares (square occupied by opponent): `{ [sq]: { background: 'radial-gradient(transparent 0%, transparent 79%, rgba(0,128,0,0.4) 80%)', borderRadius: '50%' } }` (ring highlight)
   - To detect captures: check if the square in legalMoves has an opponent piece using `gameState.game.get(square)`

e. **Chessboard props:**
