**hooks/useDailyPuzzle.ts** — Create a `'use client'` hook that:

1. Exports a discriminated union type `DailyPuzzle`:
   ```typescript
   export type DailyPuzzle =
     | { type: 'movement'; puzzle: MovementPuzzle }
     | { type: 'capture'; puzzle: CapturePuzzle };
   ```

2. Exports a pure function `getDailyPuzzle(dateStr: string): DailyPuzzle` that:
   - Combines `[...movementPuzzles.map(p => ({ type: 'movement' as const, puzzle: p })), ...capturePuzzles.map(p => ({ type: 'capture' as const, puzzle: p }))]`
   - Hashes dateStr using djb2-style polynomial hash: `hash = (hash * 31 + charCode) | 0` for each char, then `Math.abs(hash) % allPuzzles.length`
   - Returns the puzzle at that index

3. Exports a helper `getTodayUTC(): string` that returns `new Date().toISOString().split('T')[0]` (matches useStreak pattern exactly)

4. Exports `useDailyPuzzle()` hook returning `{ dateKey, dailyPuzzle, isCompleted, markCompleted }`:
   - `dateKey` = `getTodayUTC()`
   - `dailyPuzzle` = `getDailyPuzzle(dateKey)`
   - `isCompleted` state initialized from `localStorage.getItem('lepdy_chess_daily_' + dateKey) === 'true'` on mount (with try/catch, SSR guard `typeof window === 'undefined'`)
   - `markCompleted()` sets `localStorage.setItem('lepdy_chess_daily_' + dateKey, 'true')` and `setIsCompleted(true)` (with try/catch)

Storage key prefix: `lepdy_chess_daily_` (per locked decision).
