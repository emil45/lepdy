**Puzzle ordering** — build `ORDERED_PUZZLES` at module level:
```typescript
import { chessPieces } from '@/data/chessPieces';
import { movementPuzzles } from '@/data/chessPuzzles';

const PIECE_ORDER = chessPieces
  .slice()
  .sort((a, b) => a.order - b.order)
  .map(p => p.id);

const ORDERED_PUZZLES = PIECE_ORDER.flatMap(pieceId =>
  movementPuzzles
    .filter(p => p.pieceId === pieceId)
    .sort((a, b) => a.difficulty - b.difficulty)
);
```

This produces: King-1, King-2, King-3, Rook-1, ..., Pawn-3 (18 total).

**Tap handler** (`handlePuzzleSquareClick`):
1. Guard: `if (isAdvancing || isComplete) return;`
2. Guard: `if (square === puzzle.pieceSquare) return;` (tapping the piece itself is a no-op)
3. Check if `puzzle.validTargets.includes(square)`:
   - **Correct:**
     - `setFlashSquare(square); setFlashType('correct');`
     - `setShowCorrectConfetti(true);`
     - `playRandomCelebration();`
     - `setIsAdvancing(true);`
     - After 1500ms timeout: if last puzzle (puzzleIndex === ORDERED_PUZZLES.length - 1), call `completeLevel(2)`, `playSound(AudioSounds.CELEBRATION)`, `setIsComplete(true)`, then `setTimeout(onComplete, 3000)`. Otherwise, `setPuzzleIndex(prev => prev + 1)` and `resetFeedbackState()`.
   - **Wrong:**
     - `setFlashSquare(square); setFlashType('wrong');`
     - `setShowTryAgain(true);`
     - Increment wrong count: `setWrongTapCount(prev => { const next = prev + 1; if (next >= 2) setShowHints(true); return next; });`
     - After 600ms: `setFlashSquare(null); setFlashType(null);`
     - After 1200ms: `setShowTryAgain(false);`
     - Do NOT play any sound on wrong tap (per FEED-02 — no buzzer)

**squareStyles** — built with `useMemo`:
- Always highlight piece square: `{ backgroundColor: 'rgba(255, 255, 0, 0.4)' }`
- If `showHints`: green dots on each `puzzle.validTargets` square: `{ background: 'radial-gradient(circle, rgba(0,128,0,0.4) 25%, transparent 25%)', borderRadius: '50%' }`
- If `flashSquare && flashType === 'correct'`: `{ backgroundColor: 'rgba(0, 200, 0, 0.5)' }`
- If `flashSquare && flashType === 'wrong'`: `{ backgroundColor: 'rgba(255, 100, 0, 0.4)' }`
- Dependencies: `[puzzle.pieceSquare, showHints, puzzle.validTargets, flashSquare, flashType]`
