**Auth section placement:** Insert between the streak display section (ends at line ~184) and the existing `<Divider sx={{ my: 2 }} />` at line 186. The auth section replaces that divider with its own pair of dividers.

The auth section structure:
```tsx
{cloudSyncEnabled && (
  <>
    <Divider sx={{ my: 2 }} />
    <Box sx={{ width: '100%' }}>
      {loading ? (
        /* Loading state */
        <Box sx={{ display: 'flex', alignItems: 'center', gap: 1.5 }}>
          <Skeleton variant="circular" width={32} height={32} />
          <Skeleton variant="text" width={120} height={24} />
        </Box>
      ) : user ? (
        /* Signed-in state: avatar + name + sign-out button on one row */
        <Box sx={{ display: 'flex', alignItems: 'center', gap: 1.5, width: '100%' }}>
          <Avatar
            src={user.photoURL ?? undefined}
            sx={{ width: 32, height: 32 }}
          >
            {user.displayName?.[0] ?? '?'}
          </Avatar>
          <Typography
            variant="body2"
            color="secondary.main"
            sx={{ flex: 1, textAlign: direction === 'rtl' ? 'right' : 'left' }}
          >
            {user.displayName}
          </Typography>
          <Button
            variant="text"
            size="small"
            onClick={signOut}
            sx={{ color: 'secondary.main' }}
          >
            {t('home.cloudSync.signOutButton')}
          </Button>
        </Box>
      ) : (
        /* Signed-out state: Google sign-in button */
        <Box sx={{
          display: 'flex',
          flexDirection: 'column',
          alignItems: direction === 'rtl' ? 'flex-end' : 'flex-start',
          gap: 1,
          width: '100%',
        }}>
          <Button
            variant="outlined"
            size="medium"
            startIcon={<GoogleIcon />}
            onClick={handleSignIn}
            sx={{ textTransform: 'none' }}
          >
            {t('home.cloudSync.signInButton')}
          </Button>
          {signInError && (
            <Typography variant="caption" color="error">
              {t('home.cloudSync.signInTitle')}
            </Typography>
          )}
        </Box>
      )}
    </Box>
  </>
)}
```

The existing `<Divider sx={{ my: 2 }} />` at line 186 stays as-is — it now serves as the divider between the auth section (or streak section when flag is off) and the language section.

**Key implementation details:**
- `user.photoURL ?? undefined` — converts null to undefined so MUI Avatar falls back to text child (Pitfall 3 from research)
- `user.displayName?.[0] ?? '?'` — handles null displayName (Pitfall 4 from research)
- Use `home.cloudSync.signInButton` NOT `home.cloudSync.signIn` (Pitfall 1 from research)
- Use `home.cloudSync.signOutButton` NOT `home.cloudSync.signOut`
- Sign-out button uses `variant="text"` per locked decision (less prominent)
- Avatar is 32px per locked decision
- `textTransform: 'none'` on sign-in button to preserve Hebrew text casing
