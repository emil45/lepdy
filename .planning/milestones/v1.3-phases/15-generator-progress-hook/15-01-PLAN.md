    - `PuzzleProgressData`: `{ pieces: Record<string, PiecePuzzleProgress>; }` (string keys are ChessPieceId values)

    **Hook implementation — follow useChessProgress pattern exactly:**
    1. `useState<PuzzleProgressData>` initialized to `{ pieces: {} }`
    2. `useState<boolean>(false)` for `isInitialized`
    3. Import `useFeatureFlagContext` from `@/contexts/FeatureFlagContext` and destructure `getFlag`

    5. `useEffect` on `[data, isInitialized]`: if not initialized, return. try/catch `localStorage.setItem(STORAGE_KEY, JSON.stringify(data))`. Log errors.

    **SESSION-FROZEN TIER (per locked decision: "Difficulty changes between sessions only — tier locked when session starts"):**
    6. Add `useRef<Record<string, 1 | 2 | 3>>({})` named `sessionTiersRef` — this captures the tier for each piece at the moment it is first requested during this component mount (i.e., this session).
    7. `getSessionTier = useCallback((pieceId: ChessPieceId): 1 | 2 | 3 => { if (sessionTiersRef.current[pieceId] === undefined) { sessionTiersRef.current[pieceId] = (data.pieces[pieceId] ?? DEFAULT_PIECE_PROGRESS).tier; } return sessionTiersRef.current[pieceId]; }, [data.pieces])`
       - The ref captures the tier on first access per piece and returns that frozen value for the rest of the session. Since `recordCorrect`/`recordWrong` update `data` (for localStorage persistence), the actual stored tier DOES change, but `getSessionTier` returns the frozen value for the current session.
    8. Also expose `sessionTiers: sessionTiersRef.current` in the return object (read-only snapshot for debugging/display).

    **MUTATIONS:**
    9. `recordCorrect = useCallback((pieceId: ChessPieceId) => { setData(prev => { ... }) }, [getFlag])`:
       - Get `current = prev.pieces[pieceId] ?? DEFAULT_PIECE_PROGRESS`
       - `nextCorrect = current.consecutiveCorrect + 1`
       - `advanceThreshold = getFlag('chessAdvanceTierThreshold') as number`
       - `shouldAdvance = nextCorrect >= advanceThreshold`
       - Return new data with piece updated: `tier: shouldAdvance ? Math.min(3, current.tier + 1) as 1|2|3 : current.tier`, `consecutiveCorrect: shouldAdvance ? 0 : nextCorrect`, `consecutiveWrong: 0` (always reset wrong on correct)
    10. `recordWrong = useCallback((pieceId: ChessPieceId) => { setData(prev => { ... }) }, [getFlag])`:
       - Get `current = prev.pieces[pieceId] ?? DEFAULT_PIECE_PROGRESS`
       - `nextWrong = current.consecutiveWrong + 1`
       - `demoThreshold = getFlag('chessDemoTierThreshold') as number`
       - `shouldDemote = nextWrong >= demoThreshold`
       - Return: `tier: shouldDemote ? Math.max(1, current.tier - 1) as 1|2|3 : current.tier`, `consecutiveWrong: shouldDemote ? 0 : nextWrong`, `consecutiveCorrect: 0` (always reset correct on wrong)
    11. Return `{ getSessionTier, recordCorrect, recordWrong, sessionTiers: sessionTiersRef.current, data }`
