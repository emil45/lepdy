    **Core logic:**
    - `const currentPiece = chessPieces[currentIndex];` — pieces are already sorted by `order` field (INTRO-04)
    - `handleNext`: If `currentIndex < chessPieces.length - 1`, increment index. If at last piece (index 5), call `completeLevel(1)`, call `playSound(AudioSounds.CELEBRATION)`, set `isComplete = true`, then `setTimeout(() => onComplete(), 3000)`.
    - `handleBack`: If `currentIndex > 0`, decrement index. Disabled when index === 0.
    - `handlePlayAudio`: Call `playAudio(\`chess/he/${currentPiece.audioFile}\`)` — CRITICAL: use `audioFile` not `audioPath` to avoid double `/audio/` prefix.

    **Celebration screen (when `isComplete === true`):**
    - Render `<Confetti recycle={false} numberOfPieces={300} />` at top level (not nested in RTL container)

    - Large emoji or checkmark icon

    **Piece card screen (when `isComplete === false`):**
    Layout (centered, max-width 400px):
    1. **Dot progress indicator** at top: Row of 6 small circles (12x12px), filled with `primary.main` for indices <= currentIndex, `grey.300` for indices > currentIndex. Use `data-testid="progress-dots"`.
    2. **Piece card**: Box with `bgcolor: currentPiece.color`, `borderRadius: 4`, `p: 4`, centered content:
       - Large Unicode symbol: `currentPiece.symbol` in Typography with `fontSize: { xs: 96, sm: 120 }`, `lineHeight: 1`
       - Hebrew name: `t(currentPiece.translationKey)` in Typography with `variant="h3"`, `fontWeight: 'bold'`, `mt: 2`
       - Audio button: `<IconButton onClick={handlePlayAudio} data-testid="audio-button" sx={{ mt: 2, width: 64, height: 64 }}>` containing `<VolumeUpIcon sx={{ fontSize: 36 }} />`
    3. **Navigation buttons** below card: Flex row with gap, centered:
       - Back button: `<Button variant="outlined" onClick={handleBack} disabled={currentIndex === 0} startIcon={<ArrowBackIcon />}>{t('ui.back')}</Button>`
       - Next button: `<Button variant="contained" onClick={handleNext} data-testid="next-button" endIcon={<ArrowForwardIcon />}>{t('ui.next')}</Button>`
    4. **Step counter**: Typography `variant="body2"` showing `${currentIndex + 1} / ${chessPieces.length}` with `data-testid="step-counter"`
