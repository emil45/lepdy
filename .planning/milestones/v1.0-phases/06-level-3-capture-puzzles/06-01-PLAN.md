Create `app/[locale]/games/chess-game/CapturePuzzle.tsx` modeled on MovementPuzzle.tsx. Key differences from MovementPuzzle:

**Props:** Same as MovementPuzzle — `{ onComplete: () => void; completeLevel: (levelNum: number) => void }`.

**Puzzle ordering:** Sort `capturePuzzles` by difficulty ascending at module level:
```typescript
const ORDERED_PUZZLES = [...capturePuzzles].sort((a, b) => a.difficulty - b.difficulty);
```

**State machine:** Copy from MovementPuzzle verbatim: `puzzleIndex`, `wrongTapCount`, `showHints`, `flashSquare`, `flashType`, `showTryAgain`, `isAdvancing`, `isComplete`, `showCorrectConfetti`. Plus `boardWidth` with ResizeObserver pattern (containerRef, min 320, max 480).

**Click handler** (`handleSquareClick`):
- If `isAdvancing || isComplete` → return (ignore all clicks)
- If `square === puzzle.targetSquare` → return (ignore taps on the target itself)
- If `square === puzzle.correctPieceSquare` → CORRECT: setFlashSquare(square), setFlashType('correct'), setShowCorrectConfetti(true), playRandomCelebration(), setIsAdvancing(true). After 1500ms: if last puzzle → completeLevel(3), playSound(AudioSounds.CELEBRATION), setIsComplete(true), setTimeout(onComplete, 3000). Else → advance puzzleIndex, resetFeedbackState.
- If `puzzle.distractorSquares.includes(square)` → WRONG: setFlashSquare(square), setFlashType('wrong'), setShowTryAgain(true), increment wrongTapCount. If wrongTapCount >= 2 → setShowHints(true). Clear flash after 600ms, clear tryAgain after 1200ms.
- Otherwise (empty square, any other square) → return (do nothing).

**squareStyles** (useMemo):
- Always: `puzzle.targetSquare` gets red/orange ring: `{ boxShadow: 'inset 0 0 0 4px rgba(220, 60, 0, 0.85)', borderRadius: '4px' }`
- When `showHints`: `puzzle.correctPieceSquare` gets green glow: `{ boxShadow: 'inset 0 0 0 4px rgba(0, 180, 0, 0.85)', borderRadius: '4px' }`
- When `flashSquare && flashType === 'correct'`: `{ backgroundColor: 'rgba(0, 200, 0, 0.5)' }`
- When `flashSquare && flashType === 'wrong'`: `{ backgroundColor: 'rgba(255, 100, 0, 0.4)' }`

**Instruction text:** Use `t('ui.tapToCapture', { piece: t(targetPieceConfig.translationKey as Parameters<typeof t>[0]) })` where `targetPieceConfig = chessPieces.find(p => p.id === puzzle.targetPieceId)!`.

**Progress label:** Show `{puzzleIndex + 1} / {ORDERED_PUZZLES.length}` with `data-testid="puzzle-progress"`.

**Level complete screen:** Same as MovementPuzzle PLUS an extra Typography line:
```tsx
<Typography variant="h5" sx={{ textAlign: 'center', px: 2, color: 'success.main' }}>
  {t('ui.learnedChess')}
</Typography>
```

**Board rendering:** Direct `Chessboard` import from `react-chessboard` (NOT ChessBoardDynamic). Use `options` prop API with `position: puzzle.fen`, `allowDragging: false`, `onSquareClick`, `squareStyles`, `boardOrientation: 'white'`, `animationDurationInMs: 200`, `boardStyle: { width: boardWidth + 'px', maxWidth: '480px' }`. Wrap in LTR Box with containerRef exactly like MovementPuzzle.

**Imports needed:** useState, useMemo, useCallback, useRef, useEffect from react. Box, Typography from @mui/material. Chessboard from react-chessboard. useTranslations from next-intl. Confetti from react-confetti. chessPieces from @/data/chessPieces. capturePuzzles from @/data/chessPuzzles. playRandomCelebration, playSound, AudioSounds from @/utils/audio.

Do NOT use chess.js. Do NOT import ChessBoardDynamic. Trust puzzle data fields directly.
